# EE - Estilo Editorial | Portal de Noticias y Revista Digital

Este es el repositorio oficial de **EE (Estilo Editorial)**, un portal de noticias moderno y revista digital con un enfoque holístico y consciente.

## 🚀 Tecnologías Principales

- **Framework:** [Astro.js](https://astro.build/) (v5.0+)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Interfaz:** [React](https://reactjs.org/) (Formularios e interactividad)
- **Tipado:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Contenido:** Astro Content Collections (Markdown/Zod)

## ✨ Características

- 📰 **Home Page:** Diseño de periódico moderno con secciones dinámicas y rejilla adaptable.
- 📚 **Archivo de Revistas:** Visualización de portadas con integración de Flipbook interactivo.
- 🧘 **Enfoque Holístico:** Diseño minimalista y elegante centrado en la lectura y el bienestar.
- 💎 **Membresía Premium:** Sistema de gestión de contenidos exclusivos y formularios de suscripción.
- 📱 **Mobile First:** Totalmente responsivo para una lectura cómoda en cualquier dispositivo.

## 🛠️ Instalación y Desarrollo

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/pablomg-dev/entre-eras-web.git
   cd entre-eras-web
   ```

2. **Instalar dependencias (usando pnpm):**
   ```bash
   pnpm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   pnpm dev
   ```

4. **Construir para producción:**
   ```bash
   pnpm build
   ```

## 📂 Estructura del Proyecto

- `src/content/`: Definiciones de noticias y ediciones de la revista.
- `src/components/`: Componentes reutilizables (Cards, Headers, Forms).
- `src/layouts/`: Estructura base y diseño global (Layout.astro).
- `src/pages/`: Rutas del sitio y páginas dinámicas por slug.
